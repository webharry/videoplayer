<h1 align="center">videoplayer</h1>
> 通过判断浏览器终端，适配HTML5 player和Flash player，兼容IE，chrome，filefox，以及移动端主流浏览器。

## 介绍

- 使用requirejs完成js模块化。
- r.js优化页面中的js脚本和css文件，达到提高页面响应速度，减少页面所需要的http/https请求次数。
- 视频播放器主推HTML5 player，兼容flash player
